# 🚀 Personal Portfolio

Este é o repositório do meu portfólio pessoal. Uma aplicação moderna, performática e focada em apresentar soluções de engenharia, com ênfase em Back-end, Sistemas de IoT e Arquiteturas Escaláveis.

O design segue uma estética "Industrial Dark", priorizando a legibilidade de dados e uma experiência de usuário técnica e limpa.

## 🛠️ Stack Tecnológica

O projeto utiliza o que há de mais atual no ecossistema Web e de sistemas:

- **Framework:** Next.js (App Router)

- **Linguagem:** TypeScript para tipagem estrita e segurança.

- **Estilização:** Tailwind CSS com foco em componentes minimalistas e efeitos de vidro (Backdrop Blur).

- **Animações:** Framer Motion para transições suaves e modais de projetos.

- **Ícones:** Lucide React e React Icons (especialmente TbCode para a estética dev).

- **Core:** Conhecimentos aplicados de C e Node.js para a lógica de sistemas integrados.

## 📁 Estrutura do Projeto

Baseado na arquitetura modular implementada:

```
src/app/
├── components/       # Componentes globais e reutilizáveis (Header, NavItem)
├── data/             # Arquivos JSON centralizando dados de projetos e certificados
├── lib/              # Funções utilitárias (Ex: classes dinâmicas com Tailwind Merge)
└── pages/home/       # Seções principais da Single Page Application
    ├── hero-section/ # Introdução e CTA
    ├── knowledge/    # Habilidades técnicas e certificações
    └── projects/     # Grid de projetos com lógica de filtragem dinâmica
```

## 🚀 Como Executar o Projeto

1. Clone o repositório:

````
git clone [https://github.com/joaodaniel2001/portifolio-geral.git]
````

2. Instale as dependências:
````
npm install
````

3. Rode o ambiente de desenvolvimento:
````
npm run dev
````

4. Abra ``http://localhost:3000`` no seu navegador.
